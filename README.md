## Prerequisite

* [NodeJS](https://nodejs.org)
* [Yarn](https://yarnpkg.com) optional

## How to setup

You can use either `npm` or `yarn`

### `cp .env.example .env`

To create the environment variable file. Please add the `API_KEY` in newly created `.env` file.

### `npm install`

To install all the dependencies.

### `npm run build`

To build app for the production.

### `npm run server`

To run the application. Open [http://localhost:9000](http://localhost:9000) in Chrome üöÄ.
